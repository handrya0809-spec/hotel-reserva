# hotel-reserva (Opción A)


Estructura básica lista para probar localmente o subir a GitHub Pages.


## Pasos para ejecutar localmente
1. Clona o crea la carpeta del proyecto con la estructura mostrada.
2. Asegúrate de tener las imágenes en `/img`.
3. Abre `login-cliente.html` en el navegador (la app fuerza a ver el login). Si abres `index.html` sin sesión te redirige al login.
4. Para producción: sube a GitHub, activa GitHub Pages en la rama `main` (folder root).


## Notas
- `main.js` contiene la configuración de Firebase que me diste. Reemplaza si usas otra cuenta.
- Opción A no incluye roles en Firestore (básico). Si quieres roles avanzados, elige Opción B.
